Problem 3a: Model Selection Using FE-GWP1_model_selection_1.csv¶

We use the assigned dataset and perform model selection for the regression of $Y$ on the five candidate predictors:

$$ X_1,\; X_2,\; X_3,\; X_4,\; X_5. $$

To avoid trial-and-error selection, we use three justified approaches:

  1. Best subset selection over all possible non-empty models, evaluated with:
    • adjusted $R^2$,
    • AIC,
    • BIC.
  2. Forward selection using AIC.
  3. Backward selection using BIC.

We then compare the selected models and fit the final recommended regression.

The modeling problem is

$$ Y_i = \beta_0 + \beta_1 X_{1i} + \beta_2 X_{2i} + \beta_3 X_{3i} + \beta_4 X_{4i} + \beta_5 X_{5i} + \varepsilon_i, $$

but the final model does not need to include all five predictors.

response = "Y"
predictors = ["X1", "X2", "X3", "X4", "X5"]

def fit_model(cols):
    X = sm.add_constant(df[list(cols)] if len(cols) > 0 else pd.DataFrame(index=df.index))
    model = sm.OLS(df[response], X).fit()
    return model

For each candidate model, we compute:

  • adjusted $R^2$,
  • AIC,
  • BIC.

Selection rules:

  • larger adjusted $R^2$ is better,
  • smaller AIC is better,
  • smaller BIC is better.

# Forward selection using AIC
def forward_selection_aic(candidates):
    remaining = list(candidates)
    selected = []
    current_model = fit_model(selected)
    current_aic = current_model.aic
    path = []

    while remaining:
        scored = []
        for candidate in remaining:
            trial = selected + [candidate]
            model = fit_model(trial)
            scored.append((model.aic, candidate, model))

        scored.sort(key=lambda x: x[0])
        best_aic_value, best_candidate, best_model = scored[0]

        if best_aic_value < current_aic:
            selected.append(best_candidate)
            remaining.remove(best_candidate)
            current_aic = best_aic_value
            path.append({
                "Step": len(selected),
                "Added variable": best_candidate,
                "Model": tuple(selected),
                "AIC": best_model.aic,
                "BIC": best_model.bic,
                "Adj_R2": best_model.rsquared_adj
            })
        else:
            break

    final_model = fit_model(selected)
    return pd.DataFrame(path), selected, final_model

# Backward selection using BIC
def backward_selection_bic(candidates):
    selected = list(candidates)
    current_model = fit_model(selected)
    current_bic = current_model.bic
    path = []

    while len(selected) > 1:
        scored = []
        for candidate in selected:
            trial = [x for x in selected if x != candidate]
            model = fit_model(trial)
            scored.append((model.bic, candidate, model))

        scored.sort(key=lambda x: x[0])
        best_bic_value, removed_candidate, best_model = scored[0]

        if best_bic_value < current_bic:
            selected.remove(removed_candidate)
            current_bic = best_bic_value
            path.append({
                "Step": len(path) + 1,
                "Removed variable": removed_candidate,
                "Model": tuple(selected),
                "BIC": best_model.bic,
                "AIC": best_model.aic,
                "Adj_R2": best_model.rsquared_adj
            })
        else:
            break

    final_model = fit_model(selected)
    return pd.DataFrame(path), selected, final_model

Conclusion¶

Using the assigned dataset, we applied three justified model-selection approaches:

  1. Exhaustive best-subset selection, evaluated by adjusted $R^2$, AIC, and BIC.
  2. Forward selection using AIC.
  3. Backward selection using BIC.

The criteria and stepwise procedures all selected the same final subset, so the recommended regression model is strongly supported by multiple methods rather than trial-and-error selection.
